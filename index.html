A word from the developer

ContextShift started as something I built for myself. I was tired of losing track of tabs when jumping between client projects, so I built a tool that actually fixed the problem. It uses Chrome's native tab groups instead of inventing a new system, auto-saves your state, and never touches a server.

I decided to share it publicly because it might be useful to others. The Pro license is $19 once — no subscription, no renewal. I'll maintain it for as long as I can, and if it ever stops working I'll be honest about that too.

— Bruno Santos, Portugal
